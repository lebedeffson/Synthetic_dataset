# Объединенные правила предметной области

Тема: комбинированное воздействие гипертермии и ионизирующего излучения на опухолевые клетки.

Этот файл объединяет:
- ваши исходные правила из `knowledge_rules.docx`;
- литературно подтвержденные уточнения из текущей базы знаний.

## Принцип объединения

- Если ваше правило согласуется с литературой, оно сохранено.
- Если правило слишком общее, оно уточнено.
- Если правило конфликтует с более сильными данными, оно не удалено, а переведено в контекстное правило.

## Итоговые объединенные правила

### M1. Подавление репарации ДНК при нагреве

Исходные правила:
- `R1: IF temperature >= 40 THEN hr_repair = suppressed`
- `R2: IF hyperthermia = True THEN dna_repair = suppressed`

Объединенное правило:
- `ЕСЛИ температура находится в окне 41-43 C И длительность нагрева около 30-60 мин, ТО подавление репарации ДНК высокое`

Статус:
- уточнено

Причина:
- подавление репарации подтверждается не для любого нагрева вообще, а наиболее уверенно для сенсибилизирующего окна температур и достаточной thermal dose.

### M2. Рост неустраненных повреждений ДНК после RT на фоне гипертермии

Исходные правила:
- `R3: IF radiation = True AND dna_repair = suppressed THEN unrepaired_dna_damage = high`
- `R4: IF unrepaired_dna_damage = high THEN survival_fraction = low`

Объединенное правило:
- `ЕСЛИ облучение проводится при подавленной репарации ДНК, ТО уровень неустраненных повреждений ДНК высокий И доля выживших клеток снижается`

Статус:
- сохранено

### M3. Улучшение оксигенации при мягкой гипертермии

Исходные правила:
- `R9: IF hyperthermia = True THEN oxygenation = high`
- `R10: IF oxygenation = high AND radiation = True THEN radiation_damage = enhanced`

Объединенное правило:
- `ЕСЛИ гипертермия мягкая, примерно 39-42 C, особенно при исходной гипоксии опухоли, ТО оксигенация/перфузия повышается, а радиационное повреждение усиливается`

Статус:
- уточнено

Причина:
- не любая гипертермия повышает оксигенацию; при слишком высоких температурах сосудистый эффект может стать отрицательным.

### M4. Интервал между HT и RT как ключевой фактор синергии

Исходные правила:
- `R7: IF time_gap <= 1.5h THEN radiosensitization = high`
- `R8: IF time_gap > 1.5h THEN radiosensitization = reduced`

Объединенное правило:
- `ЕСЛИ интервал между гипертермией и облучением очень короткий, примерно 0-60(90) мин, ТО радиосенсибилизация высокая`
- `ЕСЛИ интервал длинный И thermal dose низкая, ТО радиосенсибилизация снижается`

Статус:
- уточнено

Причина:
- жесткий порог `1.5 h` лучше заменить нечетким правилом;
- при хорошей thermal dose эффект длинного интервала может частично компенсироваться.

### M5. Последовательность процедур сама по себе не главный фактор

Исходные правила:
- `R5: IF mode = simultaneous THEN radiosensitization = maximal`
- `R6: IF mode = sequential THEN radiosensitization = reduced`

Объединенное правило:
- `ЕСЛИ интервал короткий, ТО порядок HT before RT или RT before HT часто менее важен, чем сам интервал и достигнутая thermal dose`

Статус:
- переработано

Причина:
- правило `sequential = reduced` слишком сильное и не согласуется с частью экспериментальных данных.

### M6. Синергия комбинированного воздействия

Исходные правила:
- `R11: IF radiation = True AND hyperthermia = True THEN combined_effect = synergistic`
- `R12: IF combined_effect = synergistic THEN SF_combined < SF_radiation & < SF_hyperthermia`

Объединенное правило:
- `ЕСЛИ гипертермия и облучение применяются в сенсибилизирующем температурном окне, при коротком интервале и достаточной thermal dose, ТО комбинированный эффект синергичен`
- `ЕСЛИ эффект синергичен, ТО survival fraction после комбинированного воздействия ниже, чем после каждого воздействия по отдельности`

Статус:
- уточнено

### M7. Польза для радиорезистентных опухолевых клеток

Исходные правила:
- `R13: IF radioresistant = True AND radiation = True AND hyperthermia = True THEN benefit = high`

Объединенное правило:
- `ЕСЛИ опухоль радиорезистентна И используется сочетание RT + HT в подходящем температурно-временном режиме, ТО ожидаемая польза от добавления HT высокая`

Статус:
- сохранено с уточнением условий

### M8. Роль thermal dose и глубины опухоли

Добавлено из литературы:
- `ЕСЛИ thermal dose высокая И опухоль поверхностная И сеансы повторяются, ТО вероятность локального контроля высокая`
- `ЕСЛИ опухоль глубокая И thermal dose низкая, ТО вероятность локального контроля низкая`

Статус:
- добавлено

Причина:
- этот блок был почти не выражен в исходных правилах, но он очень важен для практической экспертной базы знаний.

### M9. Переход к высокой прямой цитотоксичности при температурах выше 43 C

Добавлено из литературы:
- `ЕСЛИ температура > 43 C И нагрев длительный, ТО прямая цитотоксичность высокая, но риск токсичности тоже высокий, а выигрыш по оксигенации снижается`

Статус:
- добавлено

## Рекомендация для экспертной системы

Для рабочей базы знаний лучше использовать трехслойное представление:
- слой 1: ваши компактные экспертные эвристики;
- слой 2: объединенные правила `M1-M9`;
- слой 3: нечеткие правила с confidence и исключениями.

Именно слой `M1-M9` лучше считать основной согласованной версией.
